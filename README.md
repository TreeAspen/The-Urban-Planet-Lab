## Getting Started

1. **Install dependencies**

   ```bash
   npm install
   ```

2. **Start the development server**

   ```bash
   npm run dev
   ```

3. **Deploy to Vercel**

   If you haven't already, install the Vercel CLI:
   ```bash
   sudo npm install -g vercel
   ```

   Then deploy the project:
   ```bash
   vercel --prod
   ```

   The project is already set up for Vercel — the CLI will automatically pick up the config from the `.vercel` directory.
